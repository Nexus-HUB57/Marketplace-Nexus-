# 📕 Playbooks · Academ'IA

> Manuais de operação passo a passo. **Print, plastifique, cole na parede.**
> Para momentos de pressão (lançamento, crise, decisão rápida).

## 🎯 Como usar

1. **Antes da crise:** leia o playbook inteiro
2. **Durante a crise:** siga o passo a passo literalmente
3. **Depois da crise:** documente o que funcionou e o que não funcionou (PR no playbook)

## 📊 Padrão de qualidade

Todo playbook segue este template:

```
# PB-NOME
> Tempo de leitura: X min · Tempo de execução: Y min

## 🚨 Quando usar
(1 parágrafo)

## ⚠️ Pré-condições
(requirements)

## 📋 Passo a passo
1. Passo
2. Passo
...

## 🛑 Quando parar e escalar
(limites de autoridade)

## 📞 Contatos de emergência
(lista)

## 📊 Pós-mortem
(o que preencher depois)
```
