# 🤝 Como convidar um afiliado para a rede

> **Tempo:** 5 min · **Nível:** Agente

## Problema

Você quer expandir sua rede MMN mas não sabe como convidar alguém oficialmente pelo painel.

## Solução

### 1. Acesse sua rede

```
Painel Afiliado → "Minha Rede" → "Convidar"
```

### 2. Escolha o método

| Método | Como |
|---|---|
| **Link de indicação** | Copie seu link único (já está no painel) |
| **Convite por e-mail** | Digite o e-mail + nome |
| **QR Code** | Gere e compartilhe em evento |

### 3. Defina o nível do convite

```yaml
level: 1      # você é sponsor direto
cycle: 14d    # janela para ativação
```

### 4. Personalize a mensagem

```
Olá [nome],

Sou [seu nome], afiliado Nexus nível [X].
Vi que você se interessa por [tema].
Posso te enviar uma condição especial?

[seu link único]
```

### 5. Acompanhe

```
Painel → Minha Rede → "Convites pendentes"
- Enviado em: [data]
- Status: aguardando cadastro
- Expira em: [X dias]
```

## Verificação (após o convite)

- [ ] Pessoa recebeu o e-mail/link
- [ ] Cadastro completado em < 14 dias
- [ ] 1ª ação dentro de 7 dias (ativação)
- [ ] Aparece na sua árvore de rede

## Dicas de ouro

- **Convide quem já te conhece** (taxa conversão 5–10× maior)
- **Mensagem personalizada > genérica** (5min de esforço, +50% reply)
- **Siga up em 3 dias** se não abriu
