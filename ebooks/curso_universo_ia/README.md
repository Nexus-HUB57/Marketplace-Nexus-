## 🛠️ Stack Padrão Recomendada (2026)

### Para Projetos com Cloud
- **LLM:** Claude Sonnet 4 / GPT-4.1
- **Framework:** LangGraph + LangSmith
- **Vector DB:** Pinecone ou Weaviate
- **Embedding:** OpenAI text-embedding-3-large + Cohere Rerank
- **Deploy:** Fly.io, Railway ou Vercel

### Para Projetos On-Prem / Soberania
- **LLM:** Llama 3.3 70B / Qwen 2.5 72B / DeepSeek V3
- **Inferência:** vLLM ou TGI
- **Runtime agent:** OpenClaw ou Claw4AI
- **Vector DB:** Qdrant self-hosted
- **Framework:** LangChain + LangGraph
- **Observability:** Langfuse self-hosted

### Para Ingestão de Documentos
- **Parser:** Docling (IBM)
- **OCR:** Docling-OCR
- **Tabelas:** TableFormer
- **Chunking:** LangChain semantic splitter
- **Embeddings:** BGE-M3 ou Voyage 3
