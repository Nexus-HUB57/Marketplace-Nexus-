**1. O que é a Coleção Técnica**

Enquanto outras coletâneas exploram o **lado filosófico, ético e operacional** da IA, a Coleção Técnica vai fundo na **engenharia**.

São ebooks para devs, arquitetos e engenheiros que querem:
- Entender como LLMs funcionam por dentro
- Projetar sistemas multi-agente robustos
- Operar IA em escala (10k+ requests/segundo)
- Resolver problemas difíceis (RAG avançado, vetor DB, federation)

---

**2. Pré-requisitos**

- Programação (Python, TypeScript ou Go)
- Conhecimento básico de LLM (o que é, como chamar API)
- Experiência com banco de dados relacional
- Docker básico
- Git

**Não exige:** PhD em ML. A coleção é prática.

**4. Como Estudar**

**Ordem sugerida:**

1. Leia o ebook 01 (Arquitetura Multi-Agente)
2. Implemente 1 dos exemplos em código
3. Meça latência, custo, qualidade
4. Otimize 1 métrica
5. Documente e compartilhe

**Tempo estimado por ebook:** 4-8h de leitura + 8-16h de prática.

# Sobre este ebook

**Arquitetura de Sistemas Multi-Agente** é o primeiro ebook da Coleção Técnica. Cobre:

- Os 4 tipos de agentes (reflex, goal-based, utility-based, learning)
- Protocolo A2A (Agent-to-Agent)
- Federation de agentes
- Observabilidade em produção
- Cases reais
